## Deploy to GitHub Pages

1. Create repo named exactly: `themahdiashrafi.github.io`
2. Push all files to `main` branch root
3. Settings → Pages → Source: Deploy from branch → main / (root)
4. Live at `https://themahdiashrafi.github.io` within ~60 seconds
